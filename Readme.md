## pinyin-shared

2020年8月17日



*基于 rust 编写实现以 `shared` 的方式编写，pinyin。*







### 编译

构建工具使用 **cargo**



cargo 工具使用

```shell
# 格式化代码
cargo fmt

# 执行测试
cargo test
```







#### 结构

程序结构参照 [deno](https://github.com/denoland/deno) 项目风格
